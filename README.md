# Just some graphics

### Compiles on:
 * Linux
 * Windows
 * Mac

## Clone (With submodules):
```
git clone --recursive git@github.com:Saggre/just-some-graphics.git
```

## Usage (Linux):
```bash
sudo apt-get install cmake libxrandr-dev libxinerama-dev libxcursor-dev libxi-dev -y
mkdir build
cd build
cmake ..
make
./main
```
